# ChatVault landing page

Static marketing page for the ChatVault Telegram bot (React + Vite + Tailwind CSS).

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Outputs static files to `dist/` — deploy to any static host (Vercel, Netlify, GitHub Pages, etc).

The bot link is set in `src/App.jsx` (`BOT_URL`).
